## 📘 PROJECT OVERVIEW

THIS REPOSITORY SERVES AS A DEEP DIVE INTO THE MECHANICS OF NEURAL NETWORKS. INSTEAD OF RELYING ON AUTOMATIC DIFFERENTIATION, THIS PROJECT MANUALLY IMPLEMENTS THE FORWARD PASS, LOSS COMPUTATION, BACKPROPAGATION CHAIN RULE, AND WEIGHT UPDATES. IT IS DESIGNED TO DEMONSTRATE MATHEMATICAL FIDELITY BY BENCHMARKING THE CUSTOM IMPLEMENTATION AGAINST TENSORFLOW/KERAS.

---

## 💬 FEATURES

- **ZERO DEEP LEARNING LIBRARIES:** BUILT STRICTLY WITH `NUMPY`, `PANDAS`, AND `MATPLOTLIB`.
- **MODULAR OOP DESIGN:** SEPARATE CLASSES FOR LAYERS, ACTIVATIONS, LOSSES, AND OPTIMIZERS.
- **CUSTOM BACKPROPAGATION:** EXACT ANALYTICAL DERIVATIVES FOR ALL MODULES COMPUTED FROM FIRST PRINCIPLES.
- **MINI-BATCH TRAINING:** IMPLEMENTED MINI-BATCH LOGIC WITH HISTORY TRACKING FOR VISUALIZATION.
- **FRAMEWORK VALIDATION:** INCLUDES NOTEBOOKS TO COMPARE SCRATCH RESULTS DIRECTLY AGAINST TENSORFLOW.

  - `UTILS.py` - DATASET GENERATION AND PLOT CONFIGURATIONS.
- **EXPERIMENTS/** – CONTAINS `TRAIN_MODEL.py` FOR THE MAIN EXECUTION LOOP.

## 🧠 MATHEMATICAL CONCEPTS IMPLEMENTED

### 1. FORWARD PROPAGATION
$$Z = W \cdot X + b$$
$$A = \sigma(Z)$$

### 2. LOSS FUNCTIONS
**MEAN SQUARED ERROR (MSE):**
$$L = \frac{1}{N} \sum_{i=1}^{N} (y_{\text{pred}}^{(i)} - y_{\text{true}}^{(i)})^2$$

**BINARY CROSS ENTROPY (BCE):**
$$L = -\frac{1}{N} \sum_{i=1}^{N} \left[ y_{\text{true}}^{(i)} \log(y_{\text{pred}}^{(i)}) + (1-y_{\text{true}}^{(i)}) \log(1-y_{\text{pred}}^{(i)}) \right]$$

### 3. BACKPROPAGATION (CHAIN RULE)
$$dW = X^T \cdot dZ$$
$$db = \sum dZ$$
$$dX = dZ \cdot W^T$$

### 4. OPTIMIZATION RULE
$$W = W - \eta \cdot dW$$
$$b = b - \eta \cdot db$$

## ⚙️ HOW IT WORKS

1.  **DATA GENERATION:** SYNTHESIZES NON-LINEAR CLASSIFICATION/REGRESSION DATA USING `UTILS.py`.
2.  **PREPROCESSING:** SPLITS DATA AND APPLIES Z-SCORE NORMALIZATION.
3.  **MODEL ARCHITECTURE:** ASSEMBLES DENSE LAYERS AND ACTIVATIONS INTO THE `NeuralNetwork` CLASS.
4.  **TRAINING LOOP:** EXECUTES MINI-BATCH GRADIENT DESCENT, TRACKING LOSS AND METRICS PER EPOCH.
5.  **EVALUATION & VISUALIZATION:** EVALUATES ON UNSEEN TEST DATA AND PLOTS CONVERGENCE CURVES.

## 🏁 USAGE

1.  **EXECUTE TERMINAL PIPELINE:** RUN `python MAIN.py` TO INITIATE DATA GENERATION, TRAINING, AND TERMINAL LOGGING.
